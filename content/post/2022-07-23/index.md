# cat

`cat` はテキストを表示するために使うコマンドです。


## オプション

### -n

行番号を表示します。


## サンプル

### file.txt　を表示する

```
$ cat file.txt
```

### file.txt　を行番号付きで表示する

```
$ cat -n file.txt
```
