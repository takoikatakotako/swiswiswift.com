# メタキャラクター

メタキャラクターは特別な意味や役割を持つ文字です。


## サンプル

### .txt拡張子を持つファイルを表示

```
$ ls *.txt
```

### mem?.md（?部分は任意の文字）を表示

```
$ ls mem?.md
```
