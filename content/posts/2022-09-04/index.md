# whereis

指定されたコマンドのバイナリー、ソース、マニュアルページの場所を表示するコマンドです。


## サンプル

### slのバイナリとマニュアルの場所を表示する

```
$ whereis sl
```

```
sl: /usr/bin/sl /usr/share/man/man1/sl.1.gz
```
