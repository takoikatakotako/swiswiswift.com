# man

コマンドのマニュアルを表示するためのコマンドです。


## オプション

### -a

すべてのセクションのmanページが表示される

### -C

設定ファイルを指定する


## サンプル

### slコマンドのマニュアルを表示する

```
$ man sl
```

### slコマンドの第一セクションのマニュアルを表示する

```
$ man 1 sl
```
